Download and Extract ZIP

Download the ZIP from GitHub.
Extract it to a folder on your computer.
Open the Folder

Open the extracted folder in VS Code or your preferred editor.
Install Node.js (if not already installed)

Download and install Node.js from nodejs.org.
Install http-server (one-time)

Open a terminal in the project folder.
Run: npm install -g http-server
Start the Local Server

In the terminal, run: http-server -p 3000
Or, if you want to use Python: python -m http.server 3000
Open the App

Go to your browser and open: http://localhost:3000
